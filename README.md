# Falcons Esports — Final Project

![Falcons Logo](img/лого.webp)

## Project Overview
This is a responsive esports team website built as a final project for frontend development.  
The site showcases the Falcons CS2 team, their players, gallery, and includes a working contact form with validation.  

## Features
- **Home Page** – Hero section and latest news.  
- **About Page** – Team history and achievements.  
- **Players Page** –  
  - Player cards with photos, nicknames, roles, K/D ratio, MVPs.  
  - View profile modal with player stats.  
  - CRUD functionality: add, edit, delete players dynamically.  
  - Search/filter players by nickname.  
- **Gallery Page** – Grid layout of images with responsive design.  
- **Contact Page** –  
  - Fully functional contact form.  
  - Validation with custom error messages.  
  - Success/failure alerts using jQuery.  

## Technologies Used
- HTML5 & CSS3  
- JavaScript & jQuery  
- Bootstrap 5  
- LocalStorage for CRUD operations  
- GitHub Pages for hosting
